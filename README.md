# mhp-algorithm-study
프로그래머스 문제를 풀이하는 알고리즘 스터디입니다.

## 📌 진행 방식
- 매주 5문제 풀이
- 각자 본인 폴더에 업로드
- 코드 리뷰 진행

## 📁 폴더 구조
- `{이름}/week01/문제파일`
  - 예: `박효정/week01/12925_문자열을_정수로_바꾸기.py`

## 📝 파일명 규칙
- `문제번호_문제명.확장자`
- 띄어쓰기는 `_`로 변환
- 문제번호는 URL의 마지막 숫자를 사용
  - 예: `12925_문자열을_정수로_바꾸기.py`

## 💾 커밋 규칙
- `solve`: 문제 풀이 추가
- `refactor`: 코드 수정
  - 예: `solve: week01 문자열을 정수로 바꾸기`
  - 예: `refactor: week01 약수의 합 개선`
